# 📱 Prácticas de SwiftUI

Repositorio con prácticas básicas desarrolladas en **SwiftUI** como parte del aprendizaje de desarrollo de aplicaciones iOS.

---

## 🚀 Objetivo

Aprender y reforzar conceptos fundamentales de SwiftUI mediante ejercicios prácticos como:

- Creación de interfaces con SwiftUI
- Manejo de estado con `@State`
- Uso de formularios (`Form`)
- Validación básica de datos de entrada
- Interacción con botones y campos de texto

---

## 🧑‍💻 Tecnologías utilizadas

- Swift 🧡
- SwiftUI 📱
- Xcode 🛠️

---

## 📂 Contenido del proyecto

Este repositorio incluye prácticas como:

- Formularios básicos
- Captura de datos del usuario
- Validación de campos (ej: solo números en edad)
- Mostrar resultados dinámicos en pantalla

---

## 🧾 Ejemplo de funcionamiento

El formulario permite:

1. Ingresar nombre
2. Ingresar edad (solo números)
3. Presionar el botón **Enviar**
4. Mostrar un mensaje personalizado
